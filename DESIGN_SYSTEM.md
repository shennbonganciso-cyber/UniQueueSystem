# UniQueue Design System

## Typography Scale
- **Page Title (H1)**: `text-2xl` (24px) - Main page headings
- **Section Heading (H2)**: `text-xl` (20px) - Section titles
- **Card Title (H3)**: `text-base` (16px) - Card headings
- **Body Text**: `text-sm` (14px) - Standard content
- **Label/Caption**: `text-xs` (12px) - Labels, captions, metadata
- **Large Display**: `text-4xl` or `text-5xl` - Queue numbers, statistics

## Spacing System (4px base unit)
- **Header**: `px-4 py-2.5`
- **Main Container**: `px-4 py-4`
- **Card Padding**: `p-4`
- **Section Margins**: `mb-4`
- **Element Gaps**: `gap-4` (large), `gap-2` (small)
- **Button Padding**: `px-4 py-2.5` (standard), `px-3 py-2` (compact)

## Icon Sizes
- **Header/Button Icons**: `w-4 h-4`
- **Card/Feature Icons**: `w-5 h-5`
- **Large Display Icons**: `w-8 h-8`

## Border Radius
- **Small**: `rounded-lg` (8px)
- **Medium**: `rounded-xl` (12px)
- **Large**: `rounded-2xl` (16px)

## Component Standards
- **Header Height**: Consistent `py-2.5` padding
- **Card Spacing**: Uniform `p-4` padding
- **Button Height**: Consistent `py-2.5` for standard buttons
- **Grid Gaps**: `gap-4` for all grid layouts
